# Редакционный центр

Редакционный центр — статический сайт редакционной политики для проектов, работающих на нескольких площадках, на русском и казахском языках. Внутри есть раздел об использовании ИИ, русскоязычная оболочка, визуальное выравнивание по AV DS 4 и локальные шрифты без внешних зависимостей.

## Что внутри

- Спокойный документный интерфейс в духе AV DS 4 со светлой темой по умолчанию
- Группировка разделов, поиск по правилам и быстрый экспорт в текстовый файл `.md`
- Статические данные без сборки и без серверных зависимостей

## Локальный запуск

Откройте `index.html` напрямую в браузере или поднимите любой статический сервер, например:

```bash
python3 -m http.server 8080
```

## Структура

- `index.html` — оболочка интерфейса и точки входа
- `styles.css` — визуальный слой
- `app.js` — рендеринг, навигация, темы, экспорт
- `data.js` — содержание разделов и метаданные
- `CONTENT_UPGRADE.md` — карта содержательной доработки и последующей чистки оболочки
