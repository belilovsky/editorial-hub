# editorial-hub

Editorial Hub — статический сайт редакционной политики для мультиплатформенных контент-проектов (RU/KZ, AI-policy), визуально выровненный по AV DS 4 без platform coupling и без внешней font-зависимости.

## Что внутри

- Спокойный AV DS 4-aligned документный интерфейс со светлой темой по умолчанию
- Группировка разделов, поиск по правилам и быстрый экспорт в Markdown
- Статические данные без сборки и без backend-зависимостей

## Локальный запуск

Откройте `index.html` напрямую в браузере или поднимите любой статический сервер, например:

```bash
python3 -m http.server 8080
```

## Структура

- `index.html` — shell и точки входа
- `styles.css` — визуальный слой
- `app.js` — рендеринг, навигация, темы, экспорт
- `data.js` — контент разделов и метаданные
- `CONTENT_UPGRADE.md` — карта контентного апгрейда и последующей чистки shell
