# Редакционный центр

Редакционный центр — статический сайт редакционной политики для проектов, работающих на нескольких площадках, на русском и казахском языках. Внутри есть раздел об использовании ИИ, русскоязычная оболочка, визуальное выравнивание по AV DS 4 и локальные шрифты без внешних зависимостей.

## Что внутри

- Спокойный документный интерфейс в духе AV DS 4 со светлой темой по умолчанию
- Группировка разделов, поиск по правилам и быстрый экспорт в текстовый файл `.md`
- Статические данные без сборки и без серверных зависимостей
- 32 раздела редакционной политики, заполненные примеры формулировок и русские подписи в интерфейсе

## Локальный запуск

Откройте `index.html` напрямую в браузере или поднимите любой статический сервер, например:

```bash
python3 -m http.server 8080
```

## Структура

- `index.html` — оболочка интерфейса и точки входа
- `styles.css` — визуальный слой
- `app.js` — рендеринг, навигация, темы, экспорт
- `data.js` — содержание разделов и метаданные
- `CONTENT_UPGRADE.md` — карта содержательной доработки и последующей чистки оболочки
- `tools/audit-content.mjs` — проверка полноты разделов и связей
- `tools/smoke-policy-shell.mjs` — контрактная проверка оболочки
- `tools/browser-smoke.mjs` — браузерная проверка маршрутов, меню и экспорта

## Проверки

```bash
node --check app.js
node --check data.js
node tools/audit-content.mjs
node tools/smoke-policy-shell.mjs
PLAYWRIGHT_MODULE=/path/to/playwright node tools/browser-smoke.mjs
```

## Продакшен

Публичный адрес: `https://edpol.qdev.run/`.

Продакшен — статическая директория `/var/www/edpol.qdev.run/` на VPS. Перед выкладкой нужно пройти проверки выше, затем синхронизировать только файлы сайта: `index.html`, `app.js`, `styles.css`, `data.js`, `robots.txt`, `sitemap.xml`, `fonts/`.

После выкладки проверяются:

- `https://edpol.qdev.run/` отдаёт «Редакционный центр»;
- `data.js` и `app.js` подключены с актуальным параметром версии;
- `/#overview`, `/#templates` и `/#sources` открываются без ошибок;
- на телефоне меню стартует свёрнутым;
- полный экспорт скачивается как `редакционный-центр-v2.0.md`.
